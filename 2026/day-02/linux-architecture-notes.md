Linux is a operating system which gives user to navigate via commands.
90% of the applications are hosted on Linux and all the public clouds are running on Linux.
Everything in a Linux as a process.
In Linux there are Directories and files a file system hierarchy.

Top 5 commands that we use:
top - to view the summary of running processes, memory, PID etc.
ls - list the directories and files in current directory.
cp - copy files and directories
mv - move files and directories.
df - to view the disk space
du - to view the disk utilization
touch - to create an empty file
ps - to view the Linux process
