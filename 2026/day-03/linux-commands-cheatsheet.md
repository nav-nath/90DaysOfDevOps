Here are the common commands used in Linux

1> ls - list the files and directories from the current directory
2> cp - copy the files and directories
3> mv - move/rename the files and directories
4> mkdir - create a new directory
5> rm - remove files/directories
6> top - provides summary of processes, memory, PID, CPU information.
7> vi - file editor in Linux
8> ps - gives running process pid, name
9> grep - help in finding the pattern in linux
10> find - finding a file or directory from file system.
11> diff - gives differences from two files
12> head - pring default top 10 lines from a file
13> tail - print default last 10 lines from a file
14> chmod - used to changed the permission of a file/directory
15> scp - copy a file from server to server.
